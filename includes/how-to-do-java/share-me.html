☕ How to Do - ShareMe ☕

❓ Description ❓

Hi everybody, how's it going? I am CSFelix and today I gonna show yall how to make a File Transfer with HTML, CSS, Giphy, JavaScript, Java and MongoDB. Don't worry about my English, I'm trynna improve it and, some day, I gonna be fluent 😂. So, let's go!







⚒️ Tools ⚒️

  • Eclipse IDE

  • TomCat 9 version or later

  • Giphy (gifs!)

  • Java 13 version or later

  • HTML, CSS and JavaScript

  • Any Browser (tested succesfully in Google Chrome, Opera, OperaGX and Edge)

  • MongoDB 4.2.2 version or later

  • MongoDB Java Driver 3.12.7 or later
