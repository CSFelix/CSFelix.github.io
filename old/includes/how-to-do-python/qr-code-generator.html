🐍 How to Do - QRCode Generator 🐍

❓ Description ❓

QR Code Generator is a Python Code that generates QR Code (😐) and insert a logo in it!! The logo's insertion is possible because the QR is generated with High Redundancy Algorithm, that allows us to cover 30% of the QR and it will still work!!!







⚒️ Tools ⚒️

  • Python 3.7 version or later;

  • Pillow package;

  • PyQRCode package




⚙️ Run ⚙️



Installing Packages



  • Open your terminal and tip these commands:


pip install pillow
pip install pyqrcode
